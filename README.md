# System-Log-Analyzer-Report-Generator

## Features

* Load and analyze log files (.txt / .log)
* Detect ERROR, WARNING, INFO logs
* Regex-based parsing
* Search logs (case-insensitive)
* Filter logs by date and level
* Generate reports (TXT / CSV)
* CLI menu-based interface

## How to Run

```bash
python log_analyzer.py
```

## Menu Options

1. Load File
2. Analyze Logs
3. Search Logs
4. Filter Logs
5. Generate Report
6. Exit

## Requirements

* Python 3.x
